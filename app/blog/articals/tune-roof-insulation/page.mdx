# Inspiration
This post was heavily inspired by [Noah Noah's Post][1] on the Tune Owners group on Facebook.

# Materials
* One Roll Reflective Insulation(48 in. x 25 ft) - [HomeDepot][2] / <ReflectiveInsulation title={'Amazon'}/>
* Outdoor Carpet(6’ wide, 20’ long) -  [HomeDepot][3] / <OutdoorCarpet title={'Amazon'}/>
* 5 cans Hi-Strength 90 Spray Adhesive - [HomeDepot][4] / <HiStrengthAdhesive title={'Amazon'}/>
* 3 Sheets Foam Insulation - [HomeDepot][5]
* One Roll Velcro - [HomeDepot][6] / <VelcroRoll title={'Amazon'}/>

# Tools
* Framing Square or Straight Edge
* Hot Knife or Knife

# Build
### Measure & Cut Foam
### Install Reflective Insulation
### Install Carpet
### Install Velcro
### Install Panels
* Measure and cut foam boards. Trace carpet and reflectix to match.
* Test fit
* Spray adhesive everything together – carpet, foam board, reflectix, and carpet
* Test fit
* Adhesive Velcro to the ceiling, attach ceiling panels

# Testing
The panels only add minimal performance increases over the stock roof in cold weather conditions. You can read more here:
[Tune Cold Wether Performance][7]
