# Inspiration
This post was heavily inspired by [Noah Noah's Post][1] on the Tune Owners group on Facebook.

I was very skeptical that these panels were going to make the camper hold heat any better. I was willing to spend the money in hopes this would help reflect heat in the summer & make the camper darker. All 3 panels together weight 20LBS and are not noticeable when manipulating the roof.

# Materials
You will see that I did not include the Reflective Insulation that Noah had on his build. I did this because I choose to use foam panels that already had a reflective backing. I was hopping this would make the build a bit cheaper with little to no performance loss.

### Install Carpet
When buying the carpet, make sure that it does not have a backing material. This will make the panels way heavier and the adhesive wont work at well!
(Add Pic of carpet)

I unrolled the carpet, laid the pru cut foam panels down supported by blocks, then sprayed the panel and slowly let it rest on the carpet. The video below shows this process.
(ADD GIF)

Once everything was bonded together, I cut the carpet to fit!
(show Picture)

### Install Velcro
Install both sides of the Velcro on the backs of the panels. I did 3 strips per panel. With the Velcro stuck together, peel the adhesive backing off of the top side of the exposed Velcro. That will stick the Velcro on the perfict place durring the first install!
(add pics)

### Install Panels
Just push and hold so the Velcro adhesive has time to bond. The Velcro makes them easily removable in the future!


# Testing
The panels only add minimal performance increases over the stock roof in cold weather conditions. You can read more here:
[Tune Cold Wether Performance][7]
The camper was noticeably darker after installing the panels. A plus if that's what you into!
